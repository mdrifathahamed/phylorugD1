# phylorugD1: Visualize Node Support Across Multiple Phylogenomic Analyses

Provides functions to visualise node support values from multiple
independent phylogenomic analyses simultaneously on a single reference
tree using the rug plot method. Although researchers routinely run
multiple phylogenomic analyses, they publish one tree and hide the rest
in supplementary materials, yet no dedicated R tool exists to visualise
node support across multiple analyses, leaving researchers unable to
communicate phylogenomic uncertainty transparently in a single
publication figure.
